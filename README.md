# openea-eap-boot
企业应用平台快速实现版EapBoot

基于Spring Boot 2.x的前后端分离企业应用平台EapBoot 
* 前端

Vue+iView Admin 

* 后端

Spring Boot 2.x/Spring Security/JWT/Spring Data JPA+Mybatis-Plus/Redis/Elasticsearch/Activiti 

分布式限流/同步锁/验证码/SnowFlake雪花算法ID生成 

动态权限管理 数据权限 工作流 代码生成 日志记录 定时任务 

第三方社交账号、短信登录 单点登录
